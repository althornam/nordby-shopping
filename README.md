# Nordby Shopping PWA

## Publiser med GitHub Pages
1. Opprett et nytt offentlig GitHub-repository, for eksempel `nordby-shopping`.
2. Last opp alle filene og mappene fra denne pakken til roten av repoet.
3. Åpne **Settings → Pages**.
4. Under **Build and deployment**, velg **Deploy from a branch**.
5. Velg `main` og `/ (root)`, og trykk **Save**.
6. GitHub viser nettadressen etter noen minutter.

## Installer på iPhone
1. Åpne GitHub Pages-adressen i Safari.
2. Trykk Del.
3. Velg **Legg til på Hjem-skjerm**.
4. Trykk **Legg til**.

Avkrysninger lagres lokalt på telefonen. Appen fungerer også offline etter første åpning.
