# Nordby Shopping PWA – flat filstruktur

Alle filer skal ligge i roten av GitHub-repositoryet.

## Publiser med GitHub Pages
1. Last opp alle filene i denne pakken.
2. Commit direkte til `main`.
3. Gå til **Settings → Pages**.
4. Velg **Deploy from a branch**.
5. Velg `main` og `/ (root)`.
6. Åpne GitHub Pages-adressen i Safari.
7. Trykk **Del → Legg til på Hjem-skjerm**.

Avkrysninger lagres lokalt på telefonen.
